* 第三方库使用、调研、二次封装

基础结构：

* 项目整体功能介绍
* 难点分析
  * 业务组件库
  * 编辑器
* 技术选型

  * 基础框架：vue/react
  * 脚手架： vue-cli
  * 打包：webpack + rollup
  * 测试工具：jest + vue-test-utils
  * 持续集成
